#### [iTerm2](https://www.iterm2.com/)
A very popular terminal app to replace MacOS Terminal. Its provide very good features like tabs, notications, etc.

#### [Oh My Zsh](https://ohmyz.sh/)
A good alternative to bash shell. It supports out-of-the-box commands and plugins that can help to boost productivity.
Please check on their [Wiki](https://github.com/ohmyzsh/ohmyzsh/wiki)

#### Zsh Plugins
* [auto-suggestions](https://github.com/zsh-users/zsh-autosuggestions):Auto suggest based on your histroy 
* [z](https://github.com/agkozak/zsh-z): Jump to a directory even if you only remember part of the directory path.

#### [Dracula Theme](https://draculatheme.com/iterm)
Make the iTerm2 more readable and can be customized to show git branches, time, etc.

#### [VSCode](https://code.visualstudio.com/)
The most popular text editor that provide plenty of extensons. It is fast.

#### VSCode Extensions 
* [TabNine](https://marketplace.visualstudio.com/items?itemName=TabNine.tabnine-vscode): Auto suggestion code based on deep learning
* [Prettier](https://github.com/prettier/prettier-vscode): Auto code formatting
* [Bracket Pair Colorizer](https://marketplace.visualstudio.com/items?itemName=CoenraadS.bracket-pair-colorizer): Colorize Each pair of Open and Close Bracket
* [Dracula Theme](https://draculatheme.com/visual-studio-code/): VSCode theme

#### [TablePlus](https://tableplus.com/)
A very lightweight and simple UI database management app. Free to use.

#### [Insonmia](https://insomnia.rest/)
A API design tool and rest client(Like Postman). Free to use.

#### [StoplightStudio](https://stoplight.io/)
Another free to use API design tool.

#### [dbdiagram.io](https://dbdiagram.io/)
A database design platform

#### [Figma](https://www.figma.com/)
UI/UX prototyping
